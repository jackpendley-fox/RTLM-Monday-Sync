How to use this app

First time? Follow these steps in order. After you’ve signed in once, you usually only need Run sync.

  1. Sign in to RTLM Click Log in to RTLM. A browser window opens — sign in with your Fox Okta account (including MFA if prompted).
  2. Wait until you’re really logged in Stay in the browser until you can see the RTLM listings (or your usual dashboard). If something looks wrong, finish the login flow before moving on.
  3. Come back to this window Switch back to this app. Don’t close the browser yet if it’s still loading — but once you’re in, return here for the next step.
  4. Save your session Click I’m logged in — save session. That stores your login so the next run doesn’t ask you to sign in again (until your session expires).
  5. Run the sync Click Run sync. The app will open RTLM, collect today’s sporting listings, and send them to Monday.com through MuleSoft. Watch the Output section below for progress. When it finishes, you’ll see a green success message and (if enabled) a short chime.

Need to stop? Use Stop if a run is stuck. Shortcuts: ⌘↵ runs sync and ⌘. stops.

Network: RTLM must be reachable (often while on VPN). If pages don’t load, check VPN and try again.

Advanced: you can also run the scraper from a terminal with npm run scrape in the scraper-bot folder — this app does the same thing with buttons.
